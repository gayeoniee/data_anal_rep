# 🛒 월마트 고객 구매 패턴 분석 및 마케팅 전략 제안

> 55만 건의 구매 데이터에서 다양한 상품군을 구매하는 핵심 고객군(692명)을 추출하고,
> 연관 구매 패턴 분석을 통해 크로스셀 전략 3가지를 도출한 프로젝트

- **데이터**: Kaggle Walmart Sales Dataset (550,068건)
- **핵심 발견**: 상품 다양성 ↔ 구매금액 상관계수 0.73
- **결과**: 마케팅 전략 3가지 제안 (묶음 상품, 테마 기획전, 추천 시스템)

## 분석 대상 주요 변수

| 변수명                | 설명                   |
| ------------------ | -------------------- |
| `User_ID`          | 고객 ID            |
| `Product_ID`       | 상품 ID              |
| `Gender`           | 성별                   |
| `Age`              | 연령대 (범주형)            |
| `Occupation`       | 직업 코드 (수치형)          |
| `City_Category`    | 도시 구분 (A, B, C)      |
| `Marital_Status`   | 결혼 여부 (0: 미혼, 1: 기혼) |
| `Product_Category` | 구매한 카테고리 번호           |
| `Purchase`         | 구매 금액                |

## 분석 흐름

1. 데이터 탐색 및 변수 정의
2. 구매금액 분포 확인 및 전처리
3. 고객별 구매 다양성 산출 (카테고리 종류 수)
4. 다양성 ↔ 구매금액 상관 분석
5. 다양성 높은 고객 정의 (15개 카테고리 이상, 692명)
6. 핵심 고객군 프로파일링 (성별, 연령, 직업, 도시, 결혼 여부)
7. 카테고리 연관 구매 분석 (상품 조합)
8. 인사이트 도출 및 마케팅 전략 제안

## 주요 분석 내용

### 구매 다양성 분석
- 고객별 구매 카테고리 종류 수 산출 → 평균 9.6개, 최대 19개
- 다양성 ↔ 총 구매금액 상관계수: **0.73 (강한 양의 상관)**
  → 다양하게 사는 고객 = 많이 쓰는 고객

### 핵심 고객군 추출
- 기준: 15개 카테고리 이상 구매 (상위 약 10%)
- 추출 결과: **692명**

### 핵심 고객군 프로파일
| 특성 | 결과 |
|------|------|
| 성별 | 남성 비율 높음 |
| 연령 | 26-35세 집중 |
| 도시 | B도시 비율 높음 |
| 결혼 | 미혼 비율 높음 |

### 연관 구매 분석
- 카테고리 1+5+8 조합: 692명 전원 구매 (가장 강한 연관성)
- 카테고리 1+5, 5+8, 1+8 쌍도 높은 동시 구매율

## 인사이트 요약

**발견 1: 다양성 = 구매력**
상품 다양성이 높은 고객은 총 구매금액도 높다 (r=0.73)
→ 다양한 카테고리를 탐색하게 만드는 것이 매출 증가의 핵심

**발견 2: 핵심 타겟은 2030 미혼 남성**
다양성 상위 10% 고객의 공통 특성: 26-35세, 남성, B도시, 미혼
→ 이 고객군에 집중한 마케팅이 효율적

**발견 3: 카테고리 1+5+8은 골든 조합**
상위 고객 692명 전원이 이 3개 카테고리를 모두 구매
→ 묶음 상품, 연관 추천의 최우선 대상


## 마케팅 전략 제안

### 1. 묶음 상품 기획
- 카테고리 1+5 / 5+8 / 1+8 조합 기반 패키지 구성
- 선택형 패키지 제공으로 유연한 구매 유도
- **기대 효과**: 객단가 상승 및 크로스셀 전환율 개선

### 2. 라이프스타일 기반 테마 기획전
- 2030 직장인, 1인 가구 등 타겟에 맞는 조합 구성
- **기대 효과**: 타겟 고객의 재방문율 및 카테고리 확장 유도

### 3. 추천 시스템 개선
- 상위 상품군 조합 기반 연관 추천 로직 설계
- **기대 효과**: 구매 전환율 및 고객 만족도 향상

## 후속 분석 제안
